# ⏱ 25 + 5 Clock (Pomodoro Timer)

A fully functional Pomodoro-style productivity timer built using React. It allows users to customize session and break lengths, start/pause the countdown, and reset the timer. Built to satisfy all freeCodeCamp user stories.

---

## 🚀 Features

- 🕐 Adjustable **Session** and **Break** durations (1–60 minutes)
- ⏯ Start / Pause toggle
- 🔁 Reset button returns everything to default
- ⏰ Countdown in **mm:ss** format
- 🔔 Audio alert plays when time reaches 00:00
- 🔄 Auto-switch between **Session** and **Break**
- 📱 Responsive and clean UI

---

## 🛠️ Technologies Used

- **React.js**
- HTML5 + CSS3
- JavaScript (ES6)
- HTML5 Audio API

## 📦 How to Run

```bash

````

2. Open `index.html` in your browser **OR** run a local server using Live Server (VS Code or Python).

## ✅ User Stories (freeCodeCamp Spec)

This project satisfies the following:

* ✅ Session/Break adjustment (1–60 mins)
* ✅ Timer countdown with display in `mm:ss` format
* ✅ Start, pause, and reset functionality
* ✅ Switch to break when session ends, and vice versa
* ✅ Audio alert (`id="beep"`) plays at 00:00
* ✅ Reset button stops timer and resets audio
